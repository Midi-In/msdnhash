About

In June 2017, the aging MSDN subscriber portal was "taken offline", with the new Visual Studio Subcription portal only being accessible to subscribers. While the old portal still works after a few DOM changes, it's not guaranteed how long it will stay up. This project aims to preserve the always publicly available MSDN hashes for reference purposes and make them easier to search with just the SHA-1 hash.

This site depends on external data, which is updated on an ad-hoc basis. There will be a delay between when new files are released by Microsoft before they show up here.
Additionally, there are certain fields that are not easily filterable with the provided data, e.g. architecture (x86/x64).
